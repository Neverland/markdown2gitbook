# auto-gitbook

## Dependence
    gitbook-cli
```
    npm install git-book -g
```
## Install

``` 
npm install auto-gitbook -g
```

## Usage

### Create

``` 
    $ auto-gitbook documentation --create
    $ auto-gitbook d -c
```

### Local preview

``` 
    $ auto-gitbook documentation --server
    $ auto-gitbook d -s
```

## Other

    1. Run `create` will auto create `_doc` dir in your `cwd`.
